Instructions - Game of Life Pattern Matching Game
• Goal: Match Target Pattern at step 12
• Actions (Keyboard Shortcuts):
  • Arrow Keys: MOVE 2x2 write window Up, Down, Left, or Right
  • Space Bar: PASS
  • Hex Keys(0-9 & A-F): WRITE a 2×2 pattern
• Update Rule: An action (move/pass/write) is always preceded by a GoL update step.
• Reset Game: Use the Reset Game button to play again.
• Save Game State as Target Pattern: Save the current Game of Life state as the new target pattern for Manual Play or Evolution Mode.
